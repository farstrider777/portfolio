About Me
Hi! I'm Josh, a developer with a big heart, a creative spirit, and a knack for solving problems. As an innovator and dreamer, I've been fascinated by computers since I was a child. When I am not at the computer I like to spend my time playing guitar, growing or gathering wild food, and gaming with my friends.

Whack A Bug
All code
A fun game project that I made one weekend and then refactored later. This project was started with a series of tests that directed the initial creation of objects and classes.

Based on the game whack-a-mole, whack-a-bug gets progressively harder each level until the user beats the game or is given his pink slip.
