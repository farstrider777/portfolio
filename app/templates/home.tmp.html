My Portfolio
Show Ranker
A website that lets users rank, review, and comment on television shows, as well as follow their friends reviews and comments. Show Ranker includes a ranking feature that lets users construct their own top shows list by presenting shows in a heads up is A better than B format.

I programmed the entire backend of this site using seqeulize and nodeJS.

Whack-A-Bug
A fun game project that I made one weekend and then refactored later. This project was started with a series of tests that directed the initial creation of objects and classes.

The game lets programmers take out their frustration on programming bugs. Based on the game whack-a-mole, whack-a-bug gets progressively harder each level until the user beats the game or is given his pink slip.
